# Portfolio 사이트

## 🔎 프로젝트 소개
HTML,CSS,JAVASCRIPT 로 포트폴리오용으로 제작한 사이트 입니다.

## 🔨 사용 기술

* `HTML`
* `CSS`
* `JavaScript`
* `Fontawesome`

## 📌 주요 기능

#### 반응형 웹 디자인을 구현하였습니다.

#### WEB API인 Intersection Observer API 를 이용하여 특정 영역 감지 시 CSS 애니메이션을 적용되게 하였습니다.

#### WEB API인 getBoundingClientRect를 이용하여 특정 영역의 높이를 계산 후 스크롤 애니메이션이 적용되게 구현하였습니다.

Github-pages 로 배포한 사이트 주소 입니다. 링크를 통해 접속하면 사이트를 직접 확인하실 수 있습니다.
